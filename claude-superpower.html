O plugin que dá superpoderes ao Claude Code.

Transforma o Claude Code de um assistente que escreve código quando você pede em um sistema que planeja antes de agir — com brainstorming, Spec, TDD e revisão automática.

Por que esse é o plugin mais importante

o que é

"Sem ele, o Claude Code tende a pular direto para a implementação — e criar problemas que só aparecem tarde demais."

Ao detectar que você está construindo algo, o Superpowers automaticamente inicia um processo de brainstorming, gera um Spec, cria um plano de implementação detalhado e só então começa a implementar.

01 · pensar

Brainstorming automático

Antes de qualquer linha de código, o sistema pensa junto com você no que precisa ser construído.

02 · planejar

Spec + plano detalhado

Gera um documento de especificação e um plano de implementação antes de começar.

03 · testar

TDD integrado

Escreve o teste antes do código. A abordagem que profissionais usam, agora automática.

04 · revisar

Revisão automática

Revisão de código e debugging estruturado entre cada etapa do desenvolvimento.

02

É seguro instalar?

segurança

Sim. O Superpowers é open source — qualquer pessoa pode ver exatamente o que ele faz antes de instalar. O repositório está público em github.com/obra/superpowers.

  • [✓]Criador conhecido. Jesse Vincent é figura pública na comunidade de desenvolvimento, com histórico verificável.
  • [✓]Não acessa a internet. O plugin funciona offline, sem enviar ou receber dados externos.
  • [✓]Não coleta dados. Nenhuma informação sua ou do seu código é enviada a terceiros.
  • [✓]Apenas instruções em markdown. O plugin é um conjunto de instruções que o Claude lê antes de agir. Simples assim.
03

Dois comandos. Pronto.

instalação

Rode os comandos abaixo dentro do Claude Code, um por vez. Depois de instalar, feche e reabra o Claude Code.

1

Registrar o repositório como fonte confiável

Este comando adiciona o marketplace do Superpowers como uma fonte de plugins reconhecida pelo Claude Code.

/plugin marketplace add obra/superpowers-marketplace
2

Instalar o Superpowers

Este comando faz a instalação do plugin em si. Execute após o primeiro ter sido concluído.

/plugin install superpowers@superpowers-marketplace
Depois de instalar, feche e reabra o Claude Code. A partir daí, o Superpowers ativa automaticamente sempre que você começar a descrever algo que quer construir.
